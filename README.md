 📊 Data analytics project 

 E-Commerce Customer Analysis Dashboard

📌 Overview

This project focuses on analyzing e-commerce customer data using Python, SQL, and Power BI to generate business insights and support data-driven decision-making. The project involves data loading, cleaning, exploratory data analysis (EDA), SQL querying, dashboard creation, and reporting.


🎯 Objectives

* Understand customer purchasing behavior.
* Identify sales trends and high-performing products.
* Analyze customer demographics and spending patterns.
* Build an interactive dashboard for business insights.

 📂 Dataset

Dataset Name: E-Commerce Customer Dataset

Contains information such as:

* Customer ID
* Customer Name
* Gender
* Age
* City
* Product Category
* Order Date
* Sales Amount
* Quantity Purchased
* Payment Method

🔄 Project Workflow

1. Data Loading

* Imported dataset using Pandas.
* Checked dataset structure and data types.

 2. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data formats.
* Converted date columns into proper format.

3. Exploratory Data Analysis (EDA)

* Customer distribution analysis.
* Sales trend analysis.
* Product category performance.
* Regional sales comparison.
* Payment method analysis.

4. SQL Analysis

Performed SQL queries to:

* Calculate total sales.
* Find top customers.
* Identify best-selling products.
* Analyze city-wise revenue.
* Generate monthly sales reports.

5. Dashboard Creation

Created an interactive Power BI dashboard containing:

* KPI Cards
* Sales Trends
* Customer Segmentation
* Product Performance
* Revenue Analysis
* Filters and Slicers

 6. Reporting & Presentation

* Prepared project report in MS Word.
* Created presentation using Gamma AI.
